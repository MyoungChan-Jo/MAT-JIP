컨테이너, 아이템, 주축(main axis), 교차축(cross axis)
컨테이너는 화물칸이고 아이템은 화물이다.
플렉스 박스는 화물칸이다 즉, div > p 일때 부모가되는 div가 flex 속성을 가지는 부모라 볼 수 있다
주축은 가로값이다. flex-direction으로 지정된 방향을 말한다 row는 가로정렬, column은 세로정렬
교차축은 세로값이다
order:0; 0은 현위치며 박스안 지정된 한개의 클래스의 순서를 바꿈
align-self: align item 을 정의한 클래스 하나에만 적용할때 사용
flex-wrap:
-nowrap: 모든 요소들을 한 줄에 정렬합니다.
-wrap: 요소들을 여러 줄에 걸쳐 정렬합니다.
-wrap-reverse: 요소들을 여러 줄에 걸쳐 반대로 정렬합니다.
flex-flow : flex-direction과flex-wrap 을 합친걸로 순서대로 기입 ex: flex-flow:column row;
align-content:
-flex-start: 여러 줄들을 컨테이너의 꼭대기에 정렬합니다.
-flex-end: 여러 줄들을 컨테이너의 바닥에 정렬합니다.
-center: 여러 줄들을 세로선 상의 가운데에 정렬합니다.
-space-between: 여러 줄들 사이에 동일한 간격을 둡니다.
-space-around: 여러 줄들 주위에 동일한 간격을 둡니다.
-stretch: 여러 줄들을 컨테이너에 맞도록 늘립니다.
